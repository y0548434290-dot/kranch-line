# Catalog Email Automation

הכנתי לך אוטומציה מלאה לקמפיין הקטלוג ולטיפול במיילי `חסום`:

- [google-apps-script-catalog-automation.gs](C:/vscode/yemot-system/docs/google-apps-script-catalog-automation.gs)

## מה זה עושה

1. שולח מייל קטלוג ראשוני מעוצב ללקוחות שיש להן מייל בשיטס.
2. מוסיף כפתור `חסום` במייל, שפותח מייל חדש עם נושא `חסום`.
3. בודק את ה־Gmail הנכנס.
4. אם נכנס מייל עם הנושא `חסום`, שולח בחזרה את הקטלוג כקובץ מצורף למייל.

## מה צריך בשיטס

הסקריפט משתמש בעמודות:

- `מייל`
- `מייל קטלוג נשלח`
- `סטטוס מייל קטלוג`

אם שתי עמודות המעקב לא קיימות, הסקריפט יוסיף אותן לבד.

## איך מתקינים

1. פתחי את [script.new](https://script.new)
2. מחקי את הקוד ברירת המחדל
3. הדביקי את התוכן של [google-apps-script-catalog-automation.gs](C:/vscode/yemot-system/docs/google-apps-script-catalog-automation.gs)
4. שמרי את הפרויקט

## איך מריצים

1. הריצי פעם אחת את `runCatalogAutomation`
2. אשרי את ההרשאות ל־Gmail, Drive ו־Sheets
3. אחר כך הריצי פעם אחת את `setupCatalogAutomationTriggers`

זה ייצור טריגר שרץ כל 5 דקות.

## איך עובד המייל הראשוני

נושא המייל:

`הפתענו!! קטלוג עטיפות קראנץ. שווה להציץ!!! יש מה לראות 😃😃😃`

בתוך המייל יש:

- טקסט שיווקי לפי הנוסח שביקשת
- כפתור לקטלוג בדרייב
- כפתור לטופס הזמנה
- כפתור `חסום`

כפתור `חסום` פותח מייל חדש עם נושא `חסום`.

## איך עובד הטיפול ב־`חסום`

הסקריפט מחפש ב־Gmail:

- מיילים שלא נקראו
- עם הנושא `חסום`
- ב־30 הימים האחרונים

כשהוא מוצא כזה:

1. הוא לוקח את קובץ הקטלוג מדרייב
2. שולח אותו חזרה כקובץ מצורף
3. מסמן את השיחה כנקראה
4. מוסיף לה תווית `kranch-catalog-blocked-processed`

כך לא יישלח שוב ושוב לאותה שיחה.
